# CAUCULADORA.PY
Código de Calculadora Simples em Python

Este código é uma calculadora simples desenvolvida em Python que permite ao usuário realizar operações matemáticas básicas pelo terminal. O programa solicita dois números e apresenta um menu com as opções de soma, subtração, multiplicação e divisão.

Após o usuário escolher a operação desejada, o código executa o cálculo correspondente e exibe o resultado na tela. No exemplo apresentado, foram digitados os números 1 e 2, escolhida a opção 1 (Soma), resultando em 3.0.
